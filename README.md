# How To Run
Go into the frontend folder:
```bash
cd frontend
```

Then install dependencies:
```bash
npm install
```

Now run Angular:
```bash
ng serve --open
```

Click on: http://localhost:4200

Open a new terminal and go to backend:
```bash
cd backend
```

Then run:
```bash
.\mvnw spring-boot:run
```

Note that you can only open the application in the frontend.

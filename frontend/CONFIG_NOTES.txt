CRUXTRACK — JSON FILES CANNOT CONTAIN COMMENTS, SO THIS FILE EXPLAINS RELATED CHANGES:

ANGULAR.JSON (SERVE SECTION):
- WE ADDED "options": { "proxyConfig": "proxy.conf.json" } SO "NG SERVE" FORWARDS /API TO THE SPRING BOOT BACKEND.
- WITHOUT THIS, THE BROWSER WOULD CALL LOCALHOST:4200/API ONLY AND THE REAL API ON PORT 8080 WOULD NOT BE REACHED (UNLESS YOU CHANGED URLs).

PROXY.CONF.JSON:
- ANY REQUEST TO HTTP://LOCALHOST:4200/API/... IS PROXIED TO HTTP://LOCALHOST:8080/API/...
- THAT LETS THE ANGULAR APP USE RELATIVE URLS LIKE "/API/AUTH/LOGIN" AND STILL HIT SPRING BOOT.
- "CHANGEORIGIN": TRUE ADJUSTS THE HOST HEADER FOR THE BACKEND (COMMON FOR DEV PROXIES).
